How to run:
1. Have go? (go version)
2. git clone https://github.com/ZoZo-182/OpenStalk.git
3. cd OpenStalk
4. go mod tidy (bubbletea, lipgloss, etc.)
5. build: go build -o OpenStalk
6. run: ./OpenStalk
